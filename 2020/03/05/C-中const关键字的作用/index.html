C++中const关键字的作用
Post：2020-03-05 22:02:31 Visit：

阅读C++ primer plus 这本书已经有一段时间，在这过程中发现const关键字非常重要，并且包含挺多细节的，最近回过头来重温，发现之前理解了的内容有一些已经遗忘了，所以决定把他们整理出来放在博客上，这样既可以与他人分享交流又方便自己日后复习、重温。我边重温边整理，所以会慢慢在这个博客中添加内容，而不会一下子总结const的所有用法，同时有错误的地方也会后面进行修改。

1、用于创建常量，使得其对应的值不能被修改，只能在初始化得时候赋值（在vs2017中规定初始化时必须赋值，否则出错）

2、修饰指针，主要有两个作用。首先是限制指针的指向，使得不能修改指针指向的变量地址，这其实跟上面的作用一样都是用于创建一个常量；第二个作用是对指针指向的变量进行保护，使得只能通过该指针读取变量的值而不能通过该指针修改变量的值。

3、const用于修饰引用(reference)。主要作用也包括两个方面，一是限制为只读，不能通过引用修改变量的值；二，const修饰的引用作为函数形参时可以在必要的时候生成临时变量。
